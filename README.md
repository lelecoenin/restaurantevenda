

# 🍽️ Dona Nena Restaurante

Projeto de website para um restaurante fictício de comida caseira.

## Funcionalidades

- Cardápio digital
- Carrinho de pedidos
- Integração com WhatsApp
- Horário de funcionamento
- Localização com mapa
- Layout responsivo
- Design moderno e intuitivo

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Desenvolver uma solução moderna para restaurantes que desejam apresentar seu cardápio online e facilitar pedidos através do WhatsApp.# restaurantevenda
